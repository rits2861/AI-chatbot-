## Project Description
This project implements an AI-powered chatbot for IT Service Management (ITSM) using Natural Language Processing (NLP). The chatbot automates common IT support tasks and reduces workload on IT staff.

## Features
- Intent Recognition using NLP
- Sentiment Analysis
- Auto Ticket Creation
- Knowledge Base Integration
- Human Escalation Support

## IT Issues Handled
- Password Reset
- VPN Issues
- Printer Problems
- Software Installation Requests
- Access Permission Requests
- Slow Computer Issues
- Ticket Status Checking

## Technologies Used
- Python 3
- Natural Language Processing (NLP)
- Intent Recognition
- Sentiment Analysis
- Seq2Seq Model Concept

## How to Run

### Step 1 - Install Python
Download Python from https://python.org

### Step 2 - Run the chatbot
```
python chatbot.py
```

### Step 3 - Chat with the bot
Type your IT issue and the bot will respond!
